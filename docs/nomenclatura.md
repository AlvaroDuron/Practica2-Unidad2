# Nomenclatura de ramas

- main: rama estable para producción.
- develop: rama principal de integración.
- feature/nombre: nuevas funcionalidades (ej. feature/login).
- bugfix/nombre: corrección de errores (ej. bugfix/registro).
- hotfix/nombre: correcciones críticas (ej. hotfix/login-crash).
- release/vX.X: preparación de versiones (ej. release/v1.0).
